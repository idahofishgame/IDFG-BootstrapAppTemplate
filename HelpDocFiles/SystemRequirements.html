GETTING STARTED: System Requirements


During creation, this application tested using the following browsers:


  • Google Chrome 36.0
  • Internet Explorer 11
  • Firefox 30.0
  • Safari 7


If you are using a different browser and have any issues, it is suggested you switch to one of the browsers listed above or newer.  Chrome and Firefox are best.
